Политика конфиденциальности

Настоящая политика описывает, как сервис Maplex обрабатывает персональные данные, которые пользователь оставляет через форму на сайте.

1. Какие данные собираются

  • имя пользователя;
  • название или тип бизнеса;
  • ссылка на карточку бизнеса, если она указана;
  • контактные данные: телефон или адрес электронной почты.

2. Для чего используются данные

  • для связи с пользователем по заявке;
  • для уточнения деталей услуги;
  • для подготовки предложения по оформлению карточки бизнеса.

3. Условия обработки

Данные обрабатываются добросовестно, в объёме, необходимом для обратной связи и оказания услуги. Сервис не передаёт информацию третьим лицам без законных оснований.

4. Защита данных

Maplex принимает разумные организационные и технические меры для защиты персональных данных от утраты, неправомерного доступа, изменения и распространения.

5. Обратная связь

По вопросам обработки данных можно обратиться по контактам, указанным на сайте. Перед публикацией проекта рекомендуется заменить демонстрационные контакты на реальные.
